# Stanza

![stanza logo](./readme-images/stanza.png)

Stanza is a classic internet forum build with Spring MVC.

## Features

- User accounts
- Admin accounts
  - Thread & post deletion
  - Add/remove admin rights
- Profile pages
  - Recent posts
  - Avatar replace
- Thread creation
  - User posts
